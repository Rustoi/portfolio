Навыки

Являюсь начинающим фронтенд-разработчиком с опытом создания статических веб-страниц.

Уверенно владею HTML, CSS и основами JavaScript.

Активно изучаю React и стремлюсь углубить свои знания в области веб-разработки.

HTML 5

"HTML — это язык разметки, используемый для создания структуры веб-страниц, позволяющий организовать текст, изображения и другие элементы контента."

CSS 3

"CSS — это язык стилей для оформления веб-страниц, позволяющий задавать цвета, шрифты и размеры, улучшая визуальное восприятие и адаптацию интерфейсов."

JavaScript

"JavaScript — это язык программирования для добавления интерактивности и динамического контента на веб-страницы, расширяющий возможности HTML и CSS."

React

"React — это библиотека JavaScript для создания динамичных пользовательских интерфейсов с использованием компонентов, упрощая разработку веб-приложений."

SaSS/SCSS

"Sass/SCSS — это препроцессор CSS, который добавляет функции, такие как переменные и вложенность, улучшая структуру и управление стилями."

Figma

"Figma — это инструмент для дизайна интерфейсов и прототипирования, позволяющий командам совместно создавать и редактировать графику."

Photoshop

"Photoshop — это мощный графический редактор для обработки изображений и создания визуального контента, широко используемый в дизайне и веб-разработке."

Redux

"Redux — это библиотека управления состоянием для JavaScript-приложений, обеспечивающая предсказуемое и централизованное управление состоянием компонентов."
